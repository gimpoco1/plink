Plink FAQ

What Is Plink?

Plink is a scorekeeping app for card games, team matches, and casual sessions where you want to save players, follow score changes, and review results later.

Can I Use It Without An Account?

Yes. Guest mode stores your game data on the current browser or device so you can start quickly. If you later want sync across devices, you can sign in and use account-backed storage.

What Kind Of Data Can I Save?

Depending on how you use the app, Plink can store session names, players, avatar colors, score history, timer settings, team assignments, and gameplay statistics.

Does Plink Support Backup And Restore?

Yes. Backup export and import tools are available so you can move game history or restore saved information when needed.

Where Can I Get Help?

Use the support page if you need help with sessions, sync, backups, or restoring data.
